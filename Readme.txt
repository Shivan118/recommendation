For first-time running:

1. Cd to the project directory
2. Create a virtual environment using requirements.txt file in the project folder
3. Activate the project virtual environment
4. Run "python setup.py" from the terminal
5. Run "python run.py" from the terminal after the setup is successfull 

For all subsequent runs:

1. Cd to the project directory
2. Activate the project virtual environment
3. Run "python run.py" from the terminal
